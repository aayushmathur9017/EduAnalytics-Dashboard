# EduAnalytics Dashboard

AI Powered Student Performance Analytics Dashboard built using Flask, SQLAlchemy, Plotly, JWT Authentication, and Docker.

## Features

- Student Performance Analytics
- AI Insights
- JWT Authentication
- Protected APIs
- CSV Upload
- PDF Export
- Docker Support
- Dark Mode
- Interactive Charts

## Tech Stack

- Python
- Flask
- SQLAlchemy
- Bootstrap
- Plotly
- JWT
- Docker

## Run Project

```bash
python app.py
